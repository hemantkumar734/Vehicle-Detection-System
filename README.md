# Vehicle Detection using Machine Learning

## Overview
An ML-based vehicle detection system built using YOLOv8, Python, OpenCV, and PyTorch. 
The system processes image frames to detect and classify vehicles using deep learning 
and computer vision techniques. This project contributed to a conference paper on 
vehicle detection and computer vision.

## Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- PyTorch
- NumPy
- Pandas

## Features
- Real-time vehicle detection from image frames
- Image preprocessing and feature extraction
- Object detection with confidence scoring
- Processed and analyzed 1,000+ image frames
- Performance optimized using GPU acceleration (CUDA)

## Project Structure
    Vehicle-Detection-System/
    ├── vehicle_detection.py   # Main detection script
    ├── requirements.txt       # Dependencies
    ├── test_images/           # Sample test images
    └── output/                # Detection output images

## How to Run

1. Clone the repository

    git clone https://github.com/hemantkumar734/Vehicle-Detection-System.git

2. Install dependencies

    pip install -r requirements.txt

3. Run detection

    python vehicle_detection.py

## Results
- Successfully detected vehicles across 1,000+ image frames
- Contributed findings to a conference paper on vehicle detection
- Achieved reliable detection using YOLOv8x model
