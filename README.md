# Vehicle Detection using Machine Learning

## Overview
An ML-based vehicle detection system built using YOLOv8, Python, OpenCV, and PyTorch. 
The system processes image frames to detect and classify vehicles using deep learning 
and computer vision techniques. This project contributed to a conference paper on 
vehicle detection and computer vision.

## Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- PyTorch
- NumPy
- Pandas

## Features
- Real-time vehicle detection from image frames
- Image preprocessing and feature extraction
- Object detection with confidence scoring
- Processed and analyzed 1,000+ image frames
- Performance optimized using GPU acceleration (CUDA)
